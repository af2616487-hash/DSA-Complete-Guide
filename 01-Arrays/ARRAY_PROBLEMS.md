# Array Practice Problems - First Year Level

## Easy Problems (Start Here!)

### 1. Sum of Array
**Problem:** Given an array, find the sum of all elements.
```
Input:  arr = [1, 2, 3, 4, 5]
Output: 15
Explanation: 1+2+3+4+5 = 15
```

### 2. Find Maximum Element
**Problem:** Find the largest element in array.
```
Input:  arr = [3, 7, 2, 9, 1]
Output: 9
```

### 3. Find Minimum Element
**Problem:** Find the smallest element in array.
```
Input:  arr = [3, 7, 2, 9, 1]
Output: 1
```

### 4. Linear Search
**Problem:** Search for a number in array. Return its index or -1 if not found.
```
Input:  arr = [10, 20, 30, 40], x = 30
Output: 2 (index of 30)
```

### 5. Reverse Array
**Problem:** Reverse the array in place.
```
Input:  arr = [1, 2, 3, 4, 5]
Output: [5, 4, 3, 2, 1]
```

### 6. Count Occurrences
**Problem:** Count how many times a number appears.
```
Input:  arr = [1, 2, 2, 3, 2, 4], x = 2
Output: 3
```

### 7. Average of Array
**Problem:** Calculate average of all elements.
```
Input:  arr = [10, 20, 30, 40]
Output: 25 (sum=100, avg=100/4=25)
```

## Medium Problems

### 8. Second Largest Element
**Problem:** Find second largest element.
```
Input:  arr = [10, 20, 30, 40, 50]
Output: 40
```

### 9. Array Rotation
**Problem:** Rotate array by k positions.
```
Input:  arr = [1,2,3,4,5], k=2
Output: [4,5,1,2,3]
```

### 10. Duplicate Check
**Problem:** Check if array has duplicates.
```
Input:  arr = [1, 2, 3, 4, 5]
Output: No

Input:  arr = [1, 2, 3, 2, 4]
Output: Yes (2 is duplicate)
```

## Tips for Solving

1. **Understand the problem** - Read carefully
2. **Trace with example** - Work through with given input
3. **Start simple** - Use loops, basic operations
4. **Test edge cases** - Empty array, single element
5. **Practice coding** - Don't just read, write code!
